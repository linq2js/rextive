# DevTools Screenshots

Place the following screenshots in this folder:

## Required Screenshots

| Filename | Description |
|----------|-------------|
| `devtools-panel.png` | Main DevTools panel overview (collapsed or hero shot) |
| `devtools-expanded.png` | DevTools panel fully expanded showing signals |
| `signals-tab.png` | Signals tab with various signal types |
| `disposed-signals.png` | Signals tab showing disposed signals (grayed out) |
| `events-tab.png` | Events tab showing event log |

## Recommended Size

- Width: 600-800px
- Format: PNG with transparency or solid background
- Style: Dark theme to match DevTools

## How to Capture

1. Run the `rextive-todo` app: `pnpm dev:todo`
2. Add some signal instances using the "useScope Test" section
3. Interact with signals to generate events
4. Remove instances to create disposed signals
5. Use browser screenshot tools or DevTools > Screenshot
